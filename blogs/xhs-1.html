UBC新生存活指南

一些血汗泪换来的实用tips

学习

1. 大一尽量不要逃课，很多基础课都是之后专业课的基础，而且很多课会有iClicker分

- 课表尽量不要排早八/晚六，因为大概率你不会去上

- 课表最好是排的紧凑一点，因为会浪费两个课中间1个小时/半个小时的空闲时间

2. 课程开始之前整理syllabus，了解一下哪些作业占比高，学会在期中/期末周prioritize

3. 无论你是哪个专业都会有属于自己的搞心态课，主要就是为了搞你心态，建议大家在入学之前跟学长学姐聊一下。遇到这种课可以稍微摆烂，因为这种课很可能是无论你努不努力都拿不了高分的课

4. 建议大家结伴学习，毕竟自己一个人很容易失去动力而且如果生病的话老师也不会update你，全靠活菩萨课友


排雷

1. 文科别报补习班，理科可以因自己需求量力而行，理科课一般补习班会有题库，和考试题目很相似，但不完全推荐大家去报，还是得靠自己学习

2. 租房的时候一定要核对好合同，检查房间/家具是否有瑕疵，核对押金等。我见过太多朋友因为这个和房东打官司的了

3. 住wesbrook和住point grey区其实差不多，wesbrook早上等车的时候也得20-30分钟，所以这边其实推荐住point grey

4. 提早申请yearround housing，建议大家接了offer之后马上就排，因为waitlist能排到你怀疑人生


娱乐

1. 多去校园走走逛逛，UBC还是挺美的

----- Wreck beach海滩

----- Rose garden春夏的时候还挺漂亮的

----- Pacific Spirit Park森林徒步

----- Botanical Garden植物园

2. 每年学费都会包括很多运动福利

----- 40🔪一个学期的健身房 ARC + Birdcoop，下午人比较多

----- Doug Mitchell免费滑冰，可以去谷歌搜一下

----- Aquatic Centre免费游泳，咱学校游泳池真的很不错，夏天的时候我会每天早上开放的时候去游几圈

----- 运动班课，如Kpop舞蹈课等，这个我个人不太了解


事业

1. 建议大一的时候多去了解一下自己未来就业方向，去LinkedIn找同校大佬

2. 大一大二多去接触一些和自己相似的同学，互相分享资源。Google/Microsoft/Meta有开放给大一大二的CS实习，可以自行了解一下

3. 大一大二多加入社团，参加一些自己感兴趣和一些能写到简历上的

4. 多提升自己的简历：CS同学可以自己做project/学leetcode等，biz的同学可以参加商赛等

5. 真心建议大家好好吃饭好好健身，会对调整自己心态有很大帮助，尤其是大一留学生，很容易因为不适应而崩溃暴饮暴食之类的

6. 别去和其他人比，因为每个人的天赋点都点的不一样（有些人是智商，有些人是社交技能，有些人是家庭背景等），所以跟其他人比没有什么参考价值
